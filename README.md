## THE PLAN 到底是个什么套餐

先把背景交代清楚。THE PLAN 是搬瓦工在2023年1月推出的一款限量版套餐，定位很明确——用限量版的价格，给你接近常规套餐两到三倍的配置。搬瓦工官方原话是这么说的：最接近的现有常规套餐（CN2 GIA-E 20G，年付169.99美元）配置其实还不如 THE PLAN，而 THE PLAN 只要99美元一年。

具体配置长这样：

- **CPU**：2核
- **内存**：2 GB
- **硬盘**：40 GB SSD RAID-10
- **月流量**：1000 GB
- **带宽**：2.5 Gbps
- **可选机房**：19个，包括中国香港HK85、日本大阪软银、洛杉矶DC6 CN2 GIA-E、DC9 CN2 GIA、圣何塞、纽约、温哥华CN2 GIA、悉尼、迪拜、阿姆斯特丹等
- **价格**：29美元/季度，或者99美元/年

这配置放在常规套餐里，差不多对标的是CN2 GIA-E 40G（3核2G/40G/2TB/2.5Gbps，年付299.99美元）。THE PLAN 核心少一个、流量少一半，但价格只有三分之一。对个人建站、科学上网、轻量应用来说，2核2G完全够用，省下来的200美元够再买两年的。

THE PLAN 之所以长期缺货，原因也简单——限量。搬瓦工的限量版套餐历来是"卖完即止，补货不定期"，官方从不提前预告具体补货时间，全靠抢。

## THE PLAN 补货规律：到底什么时候会补

先说结论：**没有固定时间表，但有几个可观察的规律**。

根据搬瓦工中文网和多个补货通知渠道的历史记录，THE PLAN 系列的补货大致呈现以下特征：

- **不定期补货**，间隔从几周到几个月不等，完全没有规律可言
- **大促节点概率较高**：黑五、双十一、618、年货节这类节点前后，搬瓦工会集中补一批限量版套餐
- **补货数量有限**：每次补货通常很快售罄，热门机房（香港HK85、DC6 CN2 GIA-E）尤其抢手
- **补货后页面会重新显示"有货"**，下单流程和常规套餐一样

所以问题就变成了：怎么在补货的瞬间就知道？这就得靠补货通知体系了。

## THE PLAN vs THE PLAN v2：多花20美元值不值

2024年搬瓦工在原版THE PLAN基础上推出了 THE PLAN v2，核心变化就一条：**流量翻倍，从1000GB涨到2000GB**，其他配置不变，价格从99美元/年涨到119美元/年（季付从29美元涨到35美元）。

搬瓦工官方对 v2 的定位说得很直白：很多用户反馈原版流量不够用，于是做了个流量翻倍版。对于跑流量较大的场景——比如代理中转、视频转发、大文件下载——v2 多出来的1000GB确实能解决"月底流量见底"的尴尬。

但如果你只是建个小站、跑个轻量服务，1000GB绰绰有余，那原版THE PLAN省下来的20美元更实在。毕竟两个套餐CPU、内存、硬盘、机房全部一样，差别只在流量。

## THE PLAN 和常规套餐比，到底便宜在哪

光看THE PLAN的99美元可能没概念，得拿常规套餐做参照。搬瓦工常规套餐分三大类：KVM入门系列、CN2 GIA-E系列、以及香港/日本/新加坡CN2 GIA系列。

THE PLAN 最直接的对手是 **CN2 GIA-E 20G**（2核1G/20G/1TB/2.5Gbps，年付169.99美元）。对比一下：

- THE PLAN 内存多1G、硬盘多20G，价格反而便宜70美元
- 两者带宽都是2.5Gbps，机房覆盖THE PLAN还更广（19个 vs 15个）

再看高一档的 **CN2 GIA-E 40G**（3核2G/40G/2TB/2.5Gbps，年付299.99美元）：

- THE PLAN 核心少1个、流量少1TB，但价格只有三分之一
- 对绝大多数个人用户，2核和3核的体感差异可以忽略

这就是THE PLAN的性价比核心：**用入门级常规套餐的价格，拿到中端常规套餐的配置**。搬瓦工自己都承认，最接近的常规套餐要299.99美元/年，而THE PLAN只要99美元。这个差价，就是限量版的价值所在。

当然，限量版的代价是"不一定买得到"。常规套餐常年有货，随买随用；THE PLAN 得等补货、得抢。这也是为什么补货通知这么重要的原因。
